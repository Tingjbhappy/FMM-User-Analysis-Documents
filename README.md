##### 用户分析报告（2019）05 ; **由于该项目为公司内部文件，作者对具体数值已经做了脱敏处理。**

#### 该项目旨在分析近期活跃用户数量未达计划预期；报告从各月MAU现状，用户拉新途径，流失用户特点，用户召回可行性，生命周期价值，以及产品影响等几方面着手分析。
